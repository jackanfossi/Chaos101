# Chaos 101
Welcome to **Chaos 101**, an interactive project developed at **Grenoble INP - Phelma** to bridge the gap between abstract mathematics and physical reality.We provide a tangible educational experience to explore how deterministic rules can lead to completely unpredictable behavior.Through our custom electronic circuit and Wi-Fi visualization system, you can discover the "order hidden behind chaos" and its vital role in modern technologies like secure data encryption.
---

## 🌀 Chaos Explained (Simply!)
If you are new to this topic, here is a simple way to understand what we are working on:

### 1. Deterministic but Unpredictable
Usually, we think the world is either **predictable**, like planetary orbits, or **random**, like thermal fluctuations. Chaos is a special third category: it follows precise rules (it is deterministic), yet it turns out to be completely unpredictable over the long term.

### 2. The Butterfly Effect
The core of chaos is **extreme sensitivity to initial conditions**. If two identical chaotic systems start from states that differ by only a microscopic margin, their trajectories will eventually yield completely different outputs. This is why even a tiny change can lead to a massive difference later on.

### 3. Why is it useful? (TRNG)
Because chaos is so sensitive and unpredictable, it is perfect for creating **True Random Numbers**. While computers usually use predictable algorithms to create "random" numbers, our circuit uses the physical "messiness" of chaos to create sequences that are mathematically impossible to predict with precision. This makes digital encryption and cybersecurity much safer.

## 📂 Project Overview: What you will find
Based on our technical specifications, this repository provides a functional educational package that includes a real working device and the associated documentation. The goal is to provide an intuitive and visual comprehension of a chaotic system through an electronic circuit.

In this project, you will find:
* **Tangible Educational Device:** A portable and interactive hardware station designed for the general public.
* **Visualization System:** Direct monitoring of the system's evolution and display of the characteristic phase space diagrams, called attractors.
* **Wireless Communication:** A microcontroller interface using Wi-Fi protocol to allow for a simple setup and smartphone connection.
* **Practical Application (TRNG):** A demo of a True Random Number Generator to showcase the relevance of chaos theory in technology and security.
* **Autonomous Power Supply:** The entire system features an autonomous power supply to ensure maximum portability for events.
* **Open-Source Access:** Full technical documentation and a guide to set up the experience autonomously.
